# Spartan-Messenger

A client server gRPC chat application
* One-one chat  
* Group chat
* LRU cache (5 messages)  
* Encrypted messages to the server  


## Steps
1. Install requirements.txt
```
pip3 install -r requirements.txt
```
2. Run server on any port
```
python3 server.py
```
3. Run the client
```
python3 client.py
```
